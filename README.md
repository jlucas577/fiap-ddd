# TravelWall

## Objetivo do Projeto
**Facilitar o planejamento de viagens.**  
“Não perca a oportunidade de ter uma viagem fantástica.”

O **TravelWall** é um projeto voltado para ajudar pessoas a planejarem melhor suas viagens, organizando ideias, decisões e estratégias de forma estruturada.
